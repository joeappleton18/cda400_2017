Setting Up Cloud 9

Cloud 9 Setup

https://c9.io/ is a feature rich cloud based programming environment. It is in effect a virtual Ubuntu linux server running a powerful text editor.

You should of all received an invite to join cloud 9, which can be found in your University email inbox.

Create A Workspace

After you sign up you will need to set up a workspace, you will only need to do this once. You should select an HTML5 workspace.

Create Your Folder Structure

It is vital that you maintain a organised file structure. It is recommended that you follow the below conventions:

  • all folder and file names should be lower case
  • names should never contain spaces
  • if you want to represent a space use a _. For example, contact us.html becomes contact_us.html
